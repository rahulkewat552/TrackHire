 # 📊 TrackHire - MERN Job Tracker

> A full-stack job application tracking system with real-time analytics dashboard.

## 🚀 Live Demo
**[https://trackhire-yy6h.onrender.com](https://trackhire-yy6h.onrender.com)**

> ⚠️ *Free tier may take 30-60 seconds to wake up.*

---

## ✨ Features

- **Authentication** - JWT with cookies
- **Dashboard** - Stats cards + Recharts (Bar + Pie)
- **Job Management** - Add, Edit, Delete
- **Search & Filter** - By status, type, and keyword
- **Responsive** - Works on all devices

---

## 🛠️ Tech Stack

**Frontend:** React, React Router, Context API, styled-components, Recharts  
**Backend:** Node.js, Express, MongoDB, Mongoose, JWT
**Deployment:** Render

---



## 📁 Structure



TrackHire/
├── client/ # React frontend
├── controllers/ # API controllers
├── models/ # MongoDB models
├── routes/ # API routes
├── middleware/ # Auth & error handling
├── utils/ # Helpers
└── server.js # Entry point
